## Rugs

### UpgradeFrom0_7_1

upgrades .rug files from the syntax used in Rug 0.7.1 to the syntax used in Rug 0.8.0

#### Running

Run it as follows:

```
$ cd to/the/repo
$ rug edit atomist-rugs:rug-upgrade:UpgradeFrom0_7_1 
```
